Cyber Safety Tips

Strong Passwords

Create long and unique passwords for every account.

Enable 2FA

Add extra security using OTP or Authenticator Apps.

Software Updates

Keep your operating system and apps updated.

Beware of Phishing

Never click suspicious links in emails or messages.

Secure Wi-Fi

Use strong Wi-Fi passwords and WPA2/WPA3 encryption.

Use Antivirus

Install trusted antivirus software and scan regularly.

Backup Data

Keep backups of important files in secure locations.

Privacy Settings

Review social media privacy settings frequently.

Download Carefully

Download apps only from trusted sources.

Lock Devices

Use PIN, Password or Biometrics on devices.

Daily Cyber Security Checklist

  • ✅ Use Strong Passwords
  • ✅ Enable Two Factor Authentication
  • ✅ Update Applications
  • ✅ Check Suspicious Emails
  • ✅ Scan For Malware
  • ✅ Backup Important Data
  • ✅ Review Account Activity
  • ✅ Use Secure Internet Connections
